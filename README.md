# GrowthFarm Mntase Site

A small static website for GrowthFarm Mntase, suitable for deployment on Vercel.

## Deploy on Vercel

1. Push this folder to GitHub.
2. In Vercel, create a new project and set the project root to `growthfarm-mntase-site`.
3. Leave the build command empty.
4. Leave the output directory empty.
5. Deploy.

The site includes:

- Home page
- Business information
- Contact email
- Privacy policy
- Terms of use

Update `hello@growthfarm.co.za` before deployment if a different public contact email should be used.
